Lab- 3: Serving Fonts Assignment



This Webpage contains the example of custom font service where we can't get the font style from the default source.

  • We can custom the font style by using @font-face rule in CSS to get font from server.

  • By specifying the location of font files stored in the server.

  • Next step is to apply the custom font to the HTML elements on webpage.
